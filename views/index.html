What is this?
This application will let you calculate either the stupidity level of a GitHub repository or the average stupidity of the top 20 repositories of a specific language, using the equation described in this article.
((Forks - Contributors) / Stars) * 100
Metrics calculated from this app should not be taken seriously, and are meant for fun.
